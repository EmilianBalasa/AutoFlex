Service Migration Tool

Warning: This tool will update all existing services in the database to include the necessary fields for search functionality. This is a one-time operation and should only be run once. Please make sure you understand what this tool does before proceeding.

This migration tool will:

  • Add location information to services that don't have it
  • Ensure price is formatted correctly as a string with currency symbol
  • Generate tags for services that don't have them
  • Set provider names for services
  • Add default images for services that don't have them

After running this tool, all services will be properly searchable using the enhanced search functionality.
